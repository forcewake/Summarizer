Summarizer
==========

Summarizer this is application which can help you with reading big boring articles. 

## Example
http://summarizer.apphb.com/ - try it!

## Uses:
* Asp.Net MVC 4
* Asp.Net WebApi
* EntityFramework
* Twitter.Bootstrap
* jQuery

## Facts
* Made with OTS.
* Support russian and english.
